README

WeatherGlass is a web-based application that provides real-time weather updates based on user input using API integration.

The system fetches weather data such as temperature, humidity, wind speed, and pressure and displays it using a clean glassmorphism interface.

Problem Statement

Users need quick and accurate weather information for planning daily activities.

This project demonstrates how real-time weather data can be fetched using APIs and presented in a simple and user-friendly interface.

Working Process

  • User enters a city name
  • Application sends request to weather API
  • API returns real-time weather data
  • JavaScript processes the response
  • UI updates dynamically
  • Weather details are displayed

Results

WeatherGlass successfully demonstrates real-time weather data retrieval and display using API integration.

The system provides accurate weather updates including temperature, humidity, wind speed, and visibility.

This project highlights frontend development, API usage, and modern UI design concepts.
